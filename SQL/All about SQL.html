SQL stands for Structureed Query Language and it is used for writing Structured Queries to a database.

This is query to find the customers with the CustomerName "Around the Horn":Select *FROM Customers HERE CustomerName = "Around The Horn"

This is a query to find all the customers who live in London. SELECT * FROM Customers WHERE City = 'London';

This is a query to find all the customers who live in the UK or in Brazil. This query would be SELECT * FROM Customers WHERE Country ="London" OR Country = "Brazil"

This is a query to find the Country which London is in. This query would be. SELECT Country FROM Customers WHERE City = "London";
